Why learn to code?

  • Perform exploratory data analysis.
  • Analyze & visualize your data.
  • Create automated reports.
  • Automate everything.
  • Make your work reproducible.
  • Create publication-quality figures.

You will want to be able to learn from others and ask for help when you need it. For scientific research, both R and Python are popular languages and great for getting started quickly with data analysis.
